# 科研记录

本项目是“科研记录”的 PySide6 源码版。软件默认运行目录为 `E:\科研记录`，业务数据保存在本机运行目录，不提交到 GitHub。

## 数据结构

新版不再使用 JSON 作为主要业务数据存储方式。首次启动时会自动创建：

```text
E:\科研记录\
├─ 数据\
│  ├─ images\
│  │  ├─ 框架图\
│  │  │  └─ 20260522_103000_框架图_001.png
│  │  ├─ 论文截图\
│  │  └─ 模型图\
│  └─ research_record.db
└─ 科研记录.exe
```

- 图片本体保存为真实图片文件，按 `images/标签名/` 自动归类。
- 图片路径以相对路径写入数据库，例如 `images/框架图/20260522_103000_框架图_001.png`。
- 标签名称、标签颜色、图片标题、备注、创建时间、更新时间等元数据保存在 SQLite 数据库 `research_record.db`。
- 数据库只保存路径和元数据，不保存图片二进制内容或 base64 文本。
- 旧版 JSON 数据只用于首次兼容读取和迁移，迁移后软件不再继续写入业务 JSON 文件。

## SQLite 表

主要表包括：

- `tags`：标签名称、颜色、创建时间、更新时间。
- `image_records`：图片标题、标签、相对图片路径、备注、链接、创建时间、更新时间。
- `settings`：必要的简单设置项，例如主题颜色和背景设置。
- `daily_tasks`、`study_sessions`、`daily_reflections`：任务、学习时长和每日心得。
- `users`：本地账号和密码哈希。

## 功能

- 登录、注册、记住账号和密码。
- 今日学习打卡、任务、每日心得。
- 任务添加、完成、删除。
- 图谱库：真实图片保存、标签文件夹归类、标签颜色、按标签筛选、图片预览、缺失图片占位提示。
- 统计分析：日历、任务完成比例、选中日期详情、学习时长趋势。
- 中文自定义主题：窗口、侧边栏、卡片、输入框、日历、文字、按钮、边框和透明度。

## 运行

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
.\.venv\Scripts\python -m research_record
```

## 仓库内容

GitHub 仓库只保存源码、依赖说明和项目文档，不保存用户真实数据、图片、数据库、exe、zip、缓存和备份文件。
